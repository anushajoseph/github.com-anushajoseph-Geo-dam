## How to configure:
**1.** Clone this repository to your local drive

```shell
$ git clone https://github.com/aswin-j/main_project
```

**2.** Installing pnpm in terminal

```shell
$ pnpm install
```

**3.** Open Terminal and Run 

```shell
$ pnpm start
```
